🔍 검사 기준

✅

안전 (0-30점)

  • 공식 기관 또는 신뢰할 수 있는 사이트
  • 유효한 SSL 인증서
  • 피싱 데이터베이스에 없음
  • 정상적인 도메인 등록 정보
⚠️

주의 (31-70점)

  • 최근 생성된 도메인
  • 의심스러운 URL 패턴
  • 일부 보안 요소 부족
  • 사용자 신고 이력 있음
🚨

위험 (71-100점)

  • 알려진 피싱 사이트
  • 악성 코드 탐지
  • 가짜 금융기관 사이트
  • 개인정보 탈취 시도

🚨 주의해야 할 URL 패턴

http://naver-security.co.kr 가짜 도메인

유명 사이트와 비슷하지만 다른 도메인을 사용

https://bit.ly/abc123 단축 URL

실제 목적지를 숨기는 단축 URL 서비스 사용

http://192.168.1.100 IP 주소

도메인 대신 IP 주소를 직접 사용

https://secure-bank-update.com 긴급성 강조

'secure', 'urgent', 'update' 등의 단어로 긴급함을 강조
